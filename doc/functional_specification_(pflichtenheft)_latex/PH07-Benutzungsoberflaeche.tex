\section{Benutzeroberfläche}

%\comment{Was sind die grundlegenden Anforderungen an die Benutzungsoberfläche (Bildschirmlayout, Dialogstruktur, ...)?}


Im Folgenden wird die grobe Struktur des grafischen Benutzeroberfläche dargestellt.
\\\\
Die Benutzeroberfläche besteht aus einem einzelnen Fenster, das verschiedene Elemente enthält.
\begin{description}
\item[Dateimenü] Das Dateimenü bietet Menüpunkte zum Anlegen eines neuen Automaten sowie zum Laden und Speichern einer Automatendefinition. Darüber hinaus kann das Programm mit einer Beenden-Menüpunkt geschlossen werden
\item[Simulationsmenü] Das Simulationsmenü bietet einen Menüpunkt um den aktuell angezeigten Automaten mit einem eingegebenen Eingabewort zu simulieren.
\item[Statusleiste] Die Statusleiste beinhaltet zusätzliche Informationen zur Simulation und zum Ergebnis.
\end{description}
