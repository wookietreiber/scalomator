\section{Glossar}

%\comment{Definition aller wichtigen Begriffe zur Sicherstellung einer einheitlichen Terminologie.}

\begin{description}
  \item[Automat] Ein Automat ist die Abstraktion eines Rechners, bestehend aus Zuständen und Zustandsübergängen.
  \item[Bibliothek] In der Programmierung ist eine Bibliothek eine Sammlung von Programmfunktionen für zusammengehörende Aufgaben. Bibliotheken sind im Allgemeinen nicht eigenständig lauffähig.
  \item[XML] (Extensible Markup Language) ist eine Auszeichnungssprache zur Darstellung hierarchisch strukturierter Daten. XML-Dateien sind Textdateien die zum Austausch von Daten verwendet werden und vom Menschen gelesen werden können.
  \item[XSD] (XML Schema Definition) definiert die Struktur eines XML-Dokuments. Die XSD-Datei ihrerseits liegt ebenfalls in Form von XML vor.
\end{description}
